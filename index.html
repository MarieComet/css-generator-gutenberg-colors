How to use this code ?

After generating the code for your colors and/or font sizes, you will need to copy the different codes and paste them in the appropriate places¬†:

  • CSS code should go into a stylesheet file, in most cases style.css file of your theme.
  • SASS code is only useful if you use it in your theme.
  • PHP code should go in the functions.php file of your theme. Learn more about theme supports in the developer documentation.
